method getStaticCache [line 278]

string getStaticCache( string $cacheName, string $generator, [int $TTL = 0])

Возвращает (если возможно - кешированный) результат выполнения указанной функции.

Функция пытается вернуть кешированный результат выполнения метода. Если это невозможно, выполняет метод и сохраняет его в кеше. В роли кешера выступает файловая система (статический кеш).




Tags:

return:  результат выполнения генератора
access:  public


Parameters:

string   $cacheName   имя метода класса генератора (по совместительству - кешированной страницы)
string   $generator   имя класса генератора
int   $TTL   время жизни кешированной страницы
