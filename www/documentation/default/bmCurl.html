method execute [line 120]

string execute( string $url, [string $method = 'GET'], [array|string $data = null])

Выполняет запрос по указанному URL Функция может выполнить как GET, так и POST запросы.

В случае, если выполняется POST запрос, в качестве данных запроса используется параметр $data. В случае, если свойства $this->emulate установлена в CURL_EMULATE_FIREFOX, то функция передает соответствующие заголовки.




Tags:

return:  результат выполнения запроса
access:  public


Parameters:

string   $url   адрес, по которому необходимо выполнить запрос
string   $method   етод запроса, может быть: "GET", "POST"
array|string   $data   данные, передаваемые в теле POST запросе
